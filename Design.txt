│     - pose_gene                                              │
│     - layout_gene                                            │
│     - style_gene                                             │
│     - lighting_gene                                          │
│     - camera_gene                                            │

│   • يثبت الـ pose/layout/depth داخل حدود الماسك             │
│   • ينتج Net (شبكة أضلاع/هيكل) كـ control image              │

│   • طبقة RGBA شفافة تُوضع كأساس إحياء جيني                 │
│   • لون أساسي خفيف (أخضر DNA-inspired) + opacity منخفض     │
└─────────────────────────────┬────────────────────────────────┘
                              │
                              ▼
┌──────────────────────────────────────────────────────────────┐
│ 5. Geometric Repair via ControlNet Net                       │
│   • يستخدم الـ Net كـ control image في عملية inpaint/img2img │
│   • يعيد بناء الشكل الهندسي والعمق بدقة عالية               │
│   • strength منخفضة (0.25–0.40) للحفاظ على الـ identity      │

                              │
                              ▼

│ 6. Colored DNA Layers (طبقات لونية مخصصة على الأضلاع)      │
│   • استخراج أضلاع الـ Net (edges/contours)                   │
│   • دمج ألوان DNA-inspired (أحمر/أخضر/أزرق) حسب:            │
│     - كثافة الماسك (density)                                 │
│     - نمط موجي (wave/helix)                                   │
│   • شفافية قابلة للتعديل                                   │
└─────────────────────────────┬────────────────────────────────┘
                              │
                              ▼
┌──────────────────────────────────────────────────────────────┐
│ 7. DNA-inspired Color Pulse (النبض الموجب/سالب)             │
│   • نبضات لونية (pixel-level حالياً، latent خيار متقدم)     │
│   • موجب: إضافة حيوية + saturation + warmth                 │
│   • سالب: قمع noise + تشوهات + تبريد مفرط                   │
│   • يطبّق فقط داخل حدود الماسك                              │
